# Release v1.0.0 - 115网盘订阅追更(PanSou版)

## 📦 版本信息
- **版本号**: v1.0.0
- **发布日期**: 2026-01-29
- **插件ID**: P115StrgmSubPansou

## ✨ 主要特性

### 1. PanSou高频搜索支持
- **解除8小时限制**: 针对PanSou搜索源，完全移除了计划任务的8小时最小间隔限制
- **灵活调度**: 用户可以设置更频繁的搜索间隔，更快地获取最新资源

### 2. 自动订阅追更
- 结合MoviePilot订阅功能，自动搜索115网盘资源
- 自动转存缺失的剧集到115网盘指定目录
- 支持电影和电视剧订阅管理

### 3. 多源搜索
- 支持PanSou、Nullbr、HDHive等多个搜索源
- 可配置优先搜索源和备用搜索源
- 智能站点切换机制

## 🔧 技术特性

### 插件标识
- **插件ID**: `P115StrgmSubPansou` (区别于原版插件)
- **命名空间**: 独立的配置和数据存储空间

### 计划任务优化
- PanSou搜索源不受8小时限制
- 其他搜索源保持原有调度逻辑
- 支持自定义cron表达式

## 📝 使用说明

1. **安装插件**: 在MoviePilot插件市场搜索"115网盘订阅追更(PanSou版)"
2. **配置115账号**: 设置115网盘的Cookie和相关参数
3. **配置搜索源**: 启用PanSou等搜索源，设置API密钥
4. **设置调度**: 配置计划任务执行周期（PanSou可设置高频）
5. **开始订阅**: 添加订阅内容，插件将自动搜索和转存

## 🎯 适用场景

- 需要快速追更的剧集订阅
- 对时效性要求高的资源获取
- PanSou重度用户

## ⚠️ 注意事项

- 高频搜索可能增加API调用次数，请注意API配额限制
- 建议合理设置搜索间隔，避免过度请求
- 确保115网盘有足够的存储空间
